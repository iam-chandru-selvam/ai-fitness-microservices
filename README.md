# AI Powered Fitness Application (Microservices)

This is a full-stack AI-powered fitness application built using **Spring Boot Microservices** and **React**.

## Tech Stack
- Spring Boot
- Spring Cloud (Eureka, Gateway, Config)
- Keycloak (Authentication)
- RabbitMQ (Async Communication)
- PostgreSQL / MySQL
- Google Gemini API
- React (Frontend)

## Microservices
- user-service
- activity-service
- (more services will be added)

## Current Status
Project is under development.  
Initial microservices setup is completed.

## How to Run
1. Clone the repository
2. Open in IntelliJ / VS Code
3. Run individual services using:
   ```bash
   mvn spring-boot:run
